Coding Lessons across South Africa

Our coding lessons take learners on a clear journey: visual block-based coding in the foundation phase, real text-based languages like Python and JavaScript in the senior phase, and portfolio-ready projects by matric. Every lesson is designed by South African educators and mapped to our Grade R–12 curriculum.
