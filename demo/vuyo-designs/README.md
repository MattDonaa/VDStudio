# Serenity Spaces - Modern Home Design Website

A modern, elegant interior design website featuring a sophisticated sage green color palette, showcasing furniture collections and design services.

## 🎨 Project Overview

Serenity Spaces is a responsive static website designed for an interior design business. The website features a clean, professional design inspired by modern home decor aesthetics with a calming sage/mint green color scheme.

## 🎨 Design System

### Color Palette
- **Primary Color**: `#a8c9c5` (Sage/Mint Green)
- **Primary Dark**: `#89b0ab`
- **Primary Light**: `#c5dbd8`
- **Secondary Color**: `#4a6362` (Dark Teal)
- **Accent Color**: `#d4a574` (Gold/Tan)
- **Background**: `#f8f9f7` (Off-white)
- **Text Dark**: `#2d3436`
- **Text Light**: `#636e72`

### Typography
- **Hero Title**: Bodoni Moda (Elegant Serif)
- **Section Headings**: Cormorant Garamond (Classic Serif)
- **Accent Text**: Playfair Display (Display Serif)
- **Decorative**: Cinzel (Luxury Serif)
- **Body Text**: Inter (Modern Sans-serif)
- **Enhanced Design**: Letter-spacing, text shadows, and custom weights for premium feel

### Responsive Breakpoints
- Desktop: 1024px+
- Tablet: 768px - 1023px
- Mobile: < 768px

## 📝 Usage Notes

### To Deploy
Please go to the **Publish tab** to deploy your website and make it live online.

### Local Development
Simply open `index.html` in a web browser to view the website locally.

### Customization
- Colors: Edit CSS variables in `:root` section of `style.css`
- Content: Modify text directly in `index.html`
- Images: Replace image URLs with your own
- Fonts: Change font imports in HTML head section
